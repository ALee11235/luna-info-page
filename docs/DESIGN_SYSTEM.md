# Design System — Luna Info Page

## Style: Dark & Luxurious

Inspired by premium fashion brands, luxury hospitality sites, and high-end SaaS marketing pages.

## Color Palette

### Primary
- `--bg-primary`: #0a0a0c (near-black, warm undertone)
- `--bg-secondary`: #12121a (dark navy-black)
- `--bg-card`: #1a1a24 (elevated surface)
- `--bg-elevated`: #22222e (higher elevation)

### Accent (Gold/Amber luxury)
- `--accent-primary`: #d4a853 (rich gold)
- `--accent-secondary`: #e8c97a (light gold)
- `--accent-muted`: #8b7355 (muted gold)
- `--accent-glow`: rgba(212, 168, 83, 0.15)

### Text
- `--text-primary`: #f5f0e8 (warm white)
- `--text-secondary`: #a8a29e (warm gray)
- `--text-muted`: #6b6560 (muted warm gray)

### Borders
- `--border-subtle`: rgba(212, 168, 83, 0.12)
- `--border-default`: rgba(212, 168, 83, 0.2)
- `--border-strong`: rgba(212, 168, 83, 0.35)

## Typography

- **Headings**: Georgia (serif, elegant)
- **Body**: Inter (clean sans-serif)
- **Tracking**: -0.02em on headings, 0.3em uppercase labels

## Key Design Principles

1. **Generous whitespace** — luxury feels spacious
2. **Subtle gold accents** — used sparingly for CTAs, highlights
3. **Warm tones** — avoid cold grays
4. **Serif headings** — elegant, editorial feel
5. **Smooth animations** — spring-based easing
6. **Mobile-first** — design for small screens first
7. **Layered depth** — subtle shadows and elevation

## Component Patterns

- **Buttons**: Gold gradient bg, dark text, glow on hover
- **Cards**: Dark bg, gold border, hover lift effect
- **Inputs**: Dark bg, gold border on focus, rounded corners
- **Sections**: 80-96px vertical padding desktop, 48-64px mobile
